Este projeto foi criado com o objetivo de praticar os conceitos básicos da linguagem Java de forma simples e prática.
O programa funciona pelo terminal e simula a criação de uma conta bancária.

Durante a execução, o usuário informa seu nome, número da conta, agência e saldo inicial. Após o preenchimento desses dados, o sistema exibe uma mensagem de boas-vindas confirmando a criação da conta com as informações digitadas.

O projeto trabalha a lógica de leitura de dados, organização do código e interação com o usuário, sendo ideal para quem está iniciando em Java e aprendendo como funciona a entrada e saída de informações pelo terminal.
